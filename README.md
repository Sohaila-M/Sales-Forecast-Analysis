# Sales & Forecast Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end ETL and Business Intelligence workflow using Python and Power BI.

The solution processes raw JSON sales and forecast datasets, transforms them into a structured dimensional model, and builds an interactive Power BI dashboard for sales and forecasting analysis.

---

# Project Components

## 1. Python ETL Script

The Python script performs:

* Data extraction from JSON files
* Data cleaning and validation
* Data transformation
* Star schema modeling
* Exporting transformed datasets as CSV files

The ETL pipeline was developed using:

* Python
* Pandas
* JSON
* Logging

---

## 2. Power BI Dashboard

The Power BI dashboard provides:

* Sales analysis
* Forecast vs Actual comparison
* Customer behavior analysis
* Product performance insights
* Time-based analysis (Year, Quarter, Month drill-down)

The dashboard is connected to the transformed CSV files generated by the ETL process.

---

# Data Model

The project uses a dimensional model consisting of:

## Fact Tables

* fact_sales
* fact_forecast

## Dimension Tables

* dim_date
* dim_year
* dim_product
* dim_customer
* dim_brand
* dim_geography

The model follows star/galaxy schema principles with one-to-many relationships.

# How to Run the ETL Script

Before running the script:

1. Update the file paths for:

   * Sales.json
   * forecast.json

2. Update the output directory path where the CSV files will be exported.

Example:

```python
sales_df = pd.DataFrame(load_json(r"YOUR_PATH/Sales.json"))
forecast_df = pd.DataFrame(load_json(r"YOUR_PATH/forecast.json"))

output_dir = "YOUR_OUTPUT_PATH"
```

3. Run the Python script.

The transformed CSV files will be generated automatically in the output folder.
